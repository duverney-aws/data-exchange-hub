KEY FLOWS:
==========

1. CMO ONBOARDING (One-time, 30 minutes):
   CMO → Portal → API Gateway → Lambda → Schema Registry + Data Contracts → Step Functions → Pipeline Deployed

2. DATA INGESTION (Ongoing, automated):
   CMO Data → Pattern 1/2/3 → Bronze Layer → Validation → Silver Layer → Aggregation → Gold Layer

3. DATA CONSUMPTION (Self-service):
   Gold Layer → Athena (SQL) → Merck Systems
   Gold Layer → Bedrock (Gen AI) → QuickSight Dashboards

4. GOVERNANCE (Continuous):
   All Data → Lake Formation (Access Control) + KMS (Encryption) + CloudTrail (Audit)


ARCHITECTURE PRINCIPLES:
========================

✓ Hub-and-Spoke: Decoupled CMOs from Merck systems
✓ Self-Service: CMOs onboard without Merck IT involvement
✓ Canonical Model: Consistent data regardless of source
✓ Medallion Architecture: Bronze (raw) → Silver (validated) → Gold (aggregated)
✓ Serverless-First: Auto-scaling, pay-per-use
✓ Security by Design: Multi-layer defense, per-CMO isolation
✓ Event-Driven: Real-time processing and notifications
✓ Immutable Audit Trail: Bronze layer never modified
